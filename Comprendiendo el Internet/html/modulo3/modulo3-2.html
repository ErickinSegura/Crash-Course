Tema 1: Tipos de Redes Introducción

Las redes se configuran con el objetivo de compartir datos de un sistema o dispositivos a otro o de disponer recursos en común, como servidores, bases de datos o impresoras, entre otros.
Son como caminos por donde viajan los datos, como enviar mensajes o compartir fotos.
En función del tamaño y del alcance de la red, se puede establecer una diferenciación entre diversas dimensiones de red. Entre los tipos de redes más importantes se encuentran:

Personal Area Networks (PAN) o red de área personal.

Es como tu círculo más cercano, donde conectas tus dispositivos en casa, como la computadora, el teléfono y la impresora. Por ejemplo, cuando envías una foto por bluetooth de una computadora a tu celular, ahí hay una red PAN.

Local Area Networks (LAN) o red de área local

Es como tu colonia, donde se conectan varias casas o edificios cercanos entre sí. En una LAN, las personas pueden compartir archivos y usar la misma impresora o internet.

Metropolitan Area Networks (MAN) o red de área metropolitana

Es como una ciudad grande, donde diferentes vecindarios se conectan entre sí. Aquí se usan para conectar edificios de una misma empresa o universidad.

Wide Area Networks (WAN) o red de área amplia

Es como un país entero, donde se conectan diferentes ciudades o regiones. Las WAN se usan para conectar sucursales de una empresa en diferentes lugares. Por ejemplo, es probable que todas las oficinas del SAT en México utilicen la misma red.

Global Area Networks (GAN) o red de área global

Es como todo el mundo conectado, donde puedes comunicarte y compartir información con personas de cualquier lugar del planeta. Por ejemplo, si alguna organización como la ONU, declarara que el internet debe ser un derecho para todos, se podría implementar una red de internet gratuita para todo el mundo.
