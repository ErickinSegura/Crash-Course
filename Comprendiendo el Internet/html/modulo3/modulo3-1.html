Módulo 3: Tipos de Redes

El término red hace referencia a un conjunto de sistemas informáticos independientes conectados entre sí, estos pueden ser una cantidad n de computadoras o dispositivos electrónicos. De tal forma que posibilitan un intercambio de datos entre todos los autorizados, para lo que es necesario tener tanto la conexión física como la conexión lógica de los sistemas. La conexión lógica se establece por medio de reglas especiales, llamadas protocolos de red especiales, como es el caso de TCP (Transmission Control Protocol), que ayuda a establecer un estándar para la entrega de datos. De ese modo, todos envían la información con el mismo formato. Dos ordenadores o dispositivos electrónicos conectados entre sí ya pueden considerarse una red, ya sea alámbrica o inalámbrica.

En este módulo se hablará de los diversos tipos de redes que existen en la actualidad y se explicará a detalle como funcionan tres de los más importantes:

  • Local Area Network (LAN)
  • Wide Area Network (WAN)
  • Virtual Private Network (VPN)
