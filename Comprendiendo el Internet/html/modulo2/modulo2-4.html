Tema 1 Navegador Web

Un navegador web, o simplemente navegador, es un programa de computadora que te permite visitar las páginas web que quieras.

Solo necesitas saber la dirección URL de la página (por ejemplo: www.google.com) o hacer clic en un enlace que te lleve a esa página, como en el enlace de azul. Algunos navegadores son:

  • Firefox
  • Edge
  • Google Chrome

Para hacer esto, se usan códigos y reglas de computadora como el HTML (HyperText Markup Language). Además, los navegadores web tienen muchas otras funciones útiles:

  • Recordar tus sitios web favoritos
  • Guardar tus contraseñas
  • Navegación privada

Resumiendo:
  • Los navegadores web son muy importantes hoy en día, ya que sin ellos no podríamos explorar el Internet.
  • Lo que hace un navegador web es conectarse a través del Internet con el servidor donde está la información que queremos, y pedirle las instrucciones para mostrar el texto, las imágenes y cómo se deben organizar para formar una página web completa.
