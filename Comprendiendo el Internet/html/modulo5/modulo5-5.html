Tema 4 Riesgos del IoT

En 2016, surgió un acontecimiento que puso de manifiesto la vulnerabilidad de los dispositivos IoT. Conocido como Mirai, este malware logró acceder a varios dispositivos utilizando las contraseñas y nombres de usuario preestablecidos de fábrica. Situaciones como estas subrayan la importancia de cambiar siempre la información predeterminada y personalizarla según nuestras necesidades.

Según Norton Security, este malware transformó los dispositivos afectados en una red de bots destinada a llevar a cabo un ataque de Denegación de Servicio Distribuido (DDoS), cuyo objetivo es saturar los sitios web con un gran volumen de tráfico de Internet. Este ataque resultó en la saturación de uno de los principales proveedores de alojamiento web del mundo, provocando la interrupción de numerosos sitios web y servicios críticos durante varias horas.
