Tema 2 ¿Qué es la web?

La World Wide Web, generalmente llamada Web para abreviar, es una colección de diferentes sitios web a los que puede acceder a través de Internet. Un sitio web está compuesto de texto relacionado, imágenes y otros recursos. Los sitios web pueden parecerse a otras formas de medios, como artículos de periódicos o programas de televisión, o pueden ser interactivos de una manera exclusiva de las computadoras. Dentro del navegador web, puedes utilizar distintos buscadores para consultar los diferentes sitios web.

El propósito de un sitio web puede ser casi cualquier cosa: una plataforma de noticias, un anuncio, una biblioteca en línea, un foro para compartir imágenes o un sitio educativo como nosotros.

Una vez que esté conectado a Internet, puede acceder y ver sitios web utilizando un tipo de aplicación llamada navegador web. Solo tenga en cuenta que el navegador web en sí no es Internet; solo muestra sitios web almacenados en Internet.

Cuando escribes una dirección web en el navegador, este envía una señal a un servidor. El servidor envía la página web al navegador, y puedes verla en tu pantalla.

Diferencias entre navegador, buscador y sitio web

Navegador Web

Es un programa o aplicación que permite acceder y navegar por sitios web. Te permite ver e interactuar con todo el contenido disponible en línea.

Buscador

Es la herramienta que te ayudará a encontrar lo que buscas. Piensa que en un libro quieres encontrar una cita importante. El buscador es la lupa que te ayudará a encontrarlo.

Sitio Web

El sitio web es aquel que contiene la información en si. En donde puedes encontrar noticias, juegos, lecturas, entre otros.
